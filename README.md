An internal web app that helps engineers draft, dedup, review, and quality-check invention patents end-to-end.

- Designed a structured **Claim Self-Check agent** (`ClaimReview`, 6 scored dimensions) with an L1 confidence gate plus an L2 "critic" pass to suppress false positives, behind a hard save-allowed / export-blocked gate.
- Shipped a **"report bug → AI triage → AI fix → human-gated merge"** GitHub Actions workflow with three layers of read-only protection.
- After an external audit scored the repo 52/100, ran a code-health sprint: deduped 14 mirror API routes behind a typed `WorkspaceScope` union, added a zod env schema, added **37 tests to a zero-test repo**, and cut a 2,953-line component by **37%**.
- **Impact:** sole engineer — **100+ merged PRs**, 6 workspace modules and an admin BI adoption dashboard over a 380-person org; drove a measurable lift in agency and patent-office approval rates (the product's north-star).

**Stack:** Next.js 14 · React 18 · TypeScript (strict) · Prisma · PostgreSQL · zod · vitest · Qwen / Qwen-VL + RAG · Docker Compose / nginx · 9 ADRs

### Self-built RTSP camera + vision-LLM workstation monitoring
A computer-vision system for factory workstation monitoring — segmenting worker actions, flagging waste / SOP violations, and raising real-time error alarms.

- Brought up a **self-built IP camera** (RTSP / H.264, 1080p25) end-to-end on macOS — bypassed a Windows-only setup tool by writing WiFi config straight to the camera's microSD, then wrote LAN auto-discovery supporting 3+ camera models.
- Fixed a real **ffmpeg 8.x** regression in the streaming service so RTSP ingestion stopped timing out.
- Built a **real-time error-prevention** feature: per-station camera binding, a continuous monitor loop with ~**5s alarm latency**, 6 new DB tables, full CRUD / REST, and **AI rule generation from SOP text or an uploaded reference video**.
- Integrated a real **Qwen-VL** vision model behind a config layer, with duplicate-segment detection to skip re-analysis.
- **Impact:** shipped V1 + V2, verified end-to-end on real hardware, iterated against graded factory-acceptance feedback.

**Stack:** Python · ffmpeg / ffprobe · Qwen-VL · custom migrations · edge / LAN deployment
